# Caso de Teste 001

## Funcionalidade
Login

## Objetivo
Validar se o usuário consegue acessar o sistema utilizando credenciais válidas.

## Pré-condições
- Usuário cadastrado.
- Página de login disponível.

## Passos

1. Acessar a página de login.
2. Informar e-mail válido.
3. Informar senha válida.
4. Clicar em "Entrar".

## Resultado Esperado

O sistema deve permitir o acesso do usuário e redirecioná-lo para a página inicial.

# Relatório de Bug 001

## Título

Botão "Entrar" não responde ao clique.

## Ambiente

Google Chrome

Windows 11

## Passos para reproduzir

1. Abrir a página de login.
2. Informar usuário válido.
3. Informar senha válida.
4. Clicar em Entrar.

## Resultado Esperado

Usuário deve acessar o sistema.

## Resultado Obtido

O botão não executa nenhuma ação.

## Prioridade

Alta
